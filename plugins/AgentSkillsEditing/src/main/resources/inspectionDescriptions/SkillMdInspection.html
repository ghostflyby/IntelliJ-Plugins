Reports missing or invalid metadata in SKILL.md files used as Codex skill definitions.

Inspects the YAML front-matter of SKILL.md files for completeness and correctness. The inspection validates that the front-matter contains the required name and description fields, that the skill name follows the kebab-case naming convention, and that the name matches the parent directory name. 


---
name: my-awesome-skill
description: A brief description of what this skill does
---

Use the provided QuickFix actions to automatically convert arbitrary names to kebab-case or rename the skill to match its parent directory name.

For structured editing support, this plugin also provides JSON Schema injection for SKILL.md front-matter, enabling code completion and inline validation in the editor.
